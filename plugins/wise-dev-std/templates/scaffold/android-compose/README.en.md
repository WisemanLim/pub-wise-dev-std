## Quick Start

```bash
# 1. Build dev flavor debug and install on connected device/emulator
make dev
```

Open in Android Studio: `File > Open > {{PROJECT_NAME}}`.

## Make Targets

| Target | Description |
|--------|-------------|
| `make dev` | Build dev debug and install (`./gradlew installDevDebug`) |
| `make test` | Run dev unit tests (`./gradlew testDevDebugUnitTest`) |
| `make build` | Build prod release bundle (`./gradlew bundleProdRelease`) |
| `make deploy` | Fastlane `beta` lane ‚Äî upload to Play Store internal track |

### Direct Gradle Usage

```bash
./gradlew tasks --all                    # list all tasks
./gradlew installDevDebug                # install on emulator/device
./gradlew testDevDebugUnitTest           # unit tests
./gradlew connectedDevDebugAndroidTest   # instrumentation tests
./gradlew bundleProdRelease              # release AAB
./gradlew lintDevDebug                   # lint
```

## Build Flavors

| Flavor | Purpose |
|--------|---------|
| `dev` | Local development / emulator |
| `staging` | QA / internal testing |
| `prod` | Play Store submission |

## Build & Deploy

### Release AAB

```bash
make build
# output: app/build/outputs/bundle/prodRelease/app-prod-release.aab
```

### Play Store internal track

```bash
make deploy
# or:
cd fastlane && bundle exec fastlane android beta
```

> **Secrets required**:
> - Keystore file (`.jks`/`.keystore`) ‚Äî **never commit**
> - `KEYSTORE_PASSWORD`, `KEY_ALIAS`, `KEY_PASSWORD` ‚Äî register as CI secrets
> - `PLAY_STORE_JSON_KEY` ‚Äî service account JSON ‚Äî register as CI secret

### CI (GitHub Actions)

Register in Repository Secrets:
- `KEYSTORE_BASE64` (keystore Base64-encoded)
- `KEYSTORE_PASSWORD`, `KEY_ALIAS`, `KEY_PASSWORD`
- `PLAY_STORE_JSON_KEY`
