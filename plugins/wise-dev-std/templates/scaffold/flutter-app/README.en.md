## Quick Start

```bash
# Install dependencies
flutter pub get

# Run on default device/emulator
make dev

# iOS simulator
make ios

# Android emulator
make android
```

## Make Targets

| Target | Description |
|--------|-------------|
| `make dev` | Run on default device (`flutter run`) |
| `make ios` | Run on iOS simulator (`flutter run -d ios`) |
| `make android` | Run on Android emulator (`flutter run -d android`) |
| `make test` | Unit/widget tests + analysis (`flutter test && flutter analyze`) |
| `make build` | Build iOS IPA + Android AAB |
| `make deploy` | Fastlane Android `beta` lane (Play internal) |

### Per-platform deployment

```bash
# iOS TestFlight:
cd fastlane && bundle exec fastlane ios beta

# Android Play Store internal:
make deploy
# or:
cd fastlane && bundle exec fastlane android beta
```

### Direct Flutter commands

```bash
flutter pub get                          # install dependencies
flutter pub upgrade                      # upgrade dependencies
flutter build ipa                        # iOS IPA
flutter build appbundle                  # Android AAB
flutter test --coverage                  # tests with coverage
flutter analyze                          # static analysis
dart format .                            # format
```

## Build & Deploy

### iOS build (macOS required)

```bash
flutter build ipa --release
# or via Fastlane:
cd fastlane && bundle exec fastlane ios beta
```

> **Secrets required**: Fastlane `match` or manual signing.  
> Never commit signing keys (.p12) or provisioning profiles.

### Android build

```bash
flutter build appbundle --release
# output: build/app/outputs/bundle/release/app-release.aab

# Upload via Fastlane:
cd fastlane && bundle exec fastlane android beta
```

> **Secrets required**: keystore (`.jks`), `PLAY_STORE_JSON_KEY` — register as CI secrets.
