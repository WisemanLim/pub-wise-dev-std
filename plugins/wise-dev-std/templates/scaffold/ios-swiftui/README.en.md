## Prerequisites

- macOS (latest stable)
- Xcode 16+ (install from App Store)
- Ruby & Bundler (`gem install bundler`)
- Fastlane (`bundle exec fastlane` or `gem install fastlane`)
- Apple Developer Program account

```bash
# Install Fastlane dependencies

```

## Quick Start

```bash

open {{PROJECT_NAME}}.xcodeproj

# 2. Build and run on simulator
make dev
```

## Make Targets

```
make <target> [SCHEME=<scheme>] [DEST=<destination>]
```

| Target | Description |
|--------|-------------|
| `make dev` | Build for simulator (default: iPhone 15) |
| `make test` | Run unit/UI tests on simulator |
| `make build` | Fastlane `build` lane — produce archive (.ipa) |
| `make deploy` | Fastlane `beta` lane — upload to TestFlight |

### Change simulator

```bash
make dev DEST='platform=iOS Simulator,name=iPhone 16 Pro'
make test SCHEME=AppTests DEST='platform=iOS Simulator,name=iPad Pro 13-inch'
```

### Direct xcodebuild

```bash
xcodebuild -scheme App \
  -destination 'platform=iOS Simulator,name=iPhone 15' \
  -configuration Debug \
  build
```

## Build & Deploy

### Archive build (local)

```bash
make build
# or:
cd fastlane && bundle exec fastlane ios build
```

### TestFlight upload

```bash
make deploy
# or:
cd fastlane && bundle exec fastlane ios beta
```

> **Secrets required**: Configure Fastlane `match` (certificates/provisioning) or manual signing.  
> Never commit signing keys (.p12) or provisioning profiles — use CI secrets / Keychain.

### CI (GitHub Actions)

- The `beta` lane in `fastlane/Fastfile` runs on a macOS runner.
- Register `APP_STORE_CONNECT_API_KEY_*` variables in Repository Secrets.
